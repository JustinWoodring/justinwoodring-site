Distributed Systems / Rust / ONNX

Slatron

Intelligent TV station manager with adaptive AI hosts and distributed playback for DIY TV stations, radio, and digital signage. Multi-provider AI DJs (Gemini, Claude, OpenAI, Ollama, local Orpheus), dynamic bumpers and station branding, spot reels, role-based access, Rhai scripting, and resilient local-cached playback. Nightly cross-architecture builds ship single self-contained binaries.

Documentation / Editor Tooling

Capibara

Crowd-sourced C library documentation and editor plugins (VS Code, Vim, Emacs). CAPI holds the human-editable YAML; Capibara Processor validates cross-references and emits a single JSON; the web API and frontend distribute it. The web layer is now Rust.

Local-First / Tabletop / Laravel + React

Lorefire

Local-first chronicle for D&D 5e campaigns. Record sessions at the table, transcribe locally with WhisperX, generate AI bardic summaries in epic prose, track characters and encounters, manage NPCs, and chat with an Oracle that has full campaign context. No cloud required. Built with NativePHP, Laravel 12, React, and Inertia.js.

Source
Quantum / Pedagogy

Qubit Simulator

Interactive 3D Bloch sphere visualization of single-qubit gate transforms — Hadamard, Pauli X/Y/Z, S, T — with animations that show how each gate moves probability amplitude. The kind of tool I wished I'd had when I was starting out.

Live demo
Electron / TypeScript / React / Rust

AutopilotV

Single-user desktop orchestrator (Electron + TypeScript) that runs two lanes in parallel: PR review in a sandboxed worktree that physically cannot push or call gh, and software development through a full task lifecycle (claim → implement → draft PR → shepherd to ready-to-merge). Pluggable adapters for trackers (Jira, GitHub Projects, Vikunja, Azure DevOps Boards), forges (GitHub, Azure DevOps), and harnesses (Claude, Codex, Cursor, OpenCode, pi). The brain is a deterministic poll loop; LLMs are called only for judgment; PR approval and merge are always an explicit human click.

Source
Agent Governance / Rust

aipolctl

Rust implementation of the Policed Agency architecture: a capability-gated policy engine for AI agents, with multi-tier escalation (executor → guardian → human) and an asymmetric communication protocol that prevents the executor from tainting the safety layer. Treats agents like employees, not root users.
